# 🚀 SwiftUI BootCamp

A hands-on SwiftUI learning repository covering core iOS development concepts and practical UI building using Apple’s SwiftUI framework.

This project is built as part of a structured bootcamp to strengthen understanding of SwiftUI fundamentals, layout systems, state management, and real-world app UI development.

---

## 📱 About the Project

This repository contains multiple SwiftUI practice projects and experiments designed to help you:

- Learn SwiftUI from basics to intermediate concepts  
- Understand declarative UI development  
- Work with layouts like `VStack`, `HStack`, and `ZStack`  
- Practice state management using `@State`, `@Binding`, and more  
- Build reusable UI components  
- Explore real-world screen designs and patterns  

---

## 🧠 What You’ll Learn

### Core SwiftUI Concepts
- Views and Modifiers
- Text, Images, Shapes
- Stacks (VStack, HStack, ZStack)
- Spacing, Padding, Alignment

### State Management
- `@State`
- `@Binding`
- `@ObservedObject`
- `@StateObject`

### Layout & UI Design
- ScrollView (Vertical & Horizontal)
- LazyStacks & LazyGrids
- NavigationStack / NavigationView
- TabView

### Advanced Topics (in progress)
- MVVM Architecture
- API Integration using URLSession
- Async/Await (Swift Concurrency)
- Reusable Components
